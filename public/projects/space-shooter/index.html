This is the arcade game inspired by the classic Space Impact.In this, player takes the control of spacecraft and has to kill enemies coming in front while surviving.

Here are the technical details of the game:

  • Implemented spacecraft movement.

  • Implemented object pooling to stop memory wastage.

  • Implemented enemy generation.
