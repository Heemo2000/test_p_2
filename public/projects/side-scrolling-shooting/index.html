Side Scrolling Shooting

Unity GIMP Personal

A shooting platformer inspired by classics Super Mario Bros and Contra.

Live Source

This is the platformer inspired by the classics Mario and Contra. In this, player has to reach the end of the level while fighting with the enemies on the way.

Here are some technical details of the project:

  • Implemented mario style jumping mechanic.

  • Implemented weapon aiming using animation rigging.

  • Implemented various types of bullets.

  • Implemented obstacles.

  • Implement enemy AI via state machine.
